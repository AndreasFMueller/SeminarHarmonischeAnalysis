Umschlagbild: Der {\em Hummelflug } ist ein orchestrales Interludium aus dem
dritten Akt der Oper {\em Das Märchen vom Zaren Saltan} von 
Nikolai Rimski-Korsakow aus dem Jahr 1899.
Das Umschlagbild zeigt das Sonogramm einer Aufnahme einer Klavierfassung
dieses Werkes.
Ein Sonogramm stellt die Intensität der zu jedem Zeitpunkt der
horizontalen Zeitachse im Signal vorhandenen Frequenzen
als helle Punkte auf der vertikalen Frequenzachse dar.
Im Kapitel~8 dieses Buches wird erklärt, wie ein Sonogramm erzeugt
wird.
